Detailed Description

Note that as with other classes that allow the user to define the extrusion's cross-sectional shape, the inherited function CgleBaseExtrusion::LoadContourPoint is re-declared as public; if it weren't, there would be no way to define the cross-section. Doxygen unfortunately insists on ignoring the 'using' keyword, and so erroneously insists LoadContourPoint is still protected.
